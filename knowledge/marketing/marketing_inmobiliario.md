# Base de Conocimiento — Marketing Inmobiliario

## Principios de Marketing

### 1. Marca Premium
- Consistencia visual
- Tono profesional
- Calidad en cada pieza
- Sin atajos

### 2. Contenido que Vende
- Enfatizar beneficios, no features
- Usar datos concretos (USD/m², m²)
- Mostrar prueba social (50% vendido)
- Crear urgencia

### 3. Plataformas
| Plataforma | Formato | Audiencia |
|------------|---------|-----------|
| WhatsApp | 750px HTML | Clientes directos |
| Instagram | 1080x1080 | Audiencia general |
| Email | 600px responsive | Leads calificados |
| Impresión | A4 300dpi | Reuniones presenciales |
