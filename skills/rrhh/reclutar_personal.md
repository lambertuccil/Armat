# Skill: reclutar_personal

## Identidad
- **Agente**: RRHH
- **Nombre**: Reclutamiento de Personal
- **Versión**: 1.0

## Objetivo
Reclutar y seleccionar personal calificado para puestos vacantes de ARMAT.

## Entradas
- **puesto** (requerido): Nombre del puesto
- **departamento** (requerido): Departamento solicitante
- **requisitos** (requerido): Experiencia, habilidades, estudios
- **sueldo** (requerido): Rango salarial

## Proceso
1. Definir perfil del candidato
2. Publicar vacante
3. Recibir CVs
4. Preseleccionar
5. Entrevistar
6. Verificar referencias
7. Hacer oferta

## Salida
Ficha de contratación con:
- Candidato seleccionado
- Perfil evaluado
- Oferta salarial
- Fecha de ingreso

## Checklist
- [ ] Perfil definido
- [ ] Vacante publicada
- [ ] CVs evaluados
- [ ] Entrevistas realizadas
- [ ] Referencias verificadas
- [ ] Oferta enviada

## Ejemplo
```markdown
# VACANCIA #2026-012

## Puesto: Técnico en Plomería
## Sueldo: $18,000/mes

## Candidatos
1. Juan García (8/10) - 3 años exp
2. Pedro López (7/10) - 2 años exp
3. Carlos Ruiz (6/10) - 1 año exp

## Recomendación: Juan García
- Experiencia sólida
- Referencias positivas
- Disponible inmediato
```
