# Skill: evaluar_desempeno

## Identidad
- **Agente**: RRHH
- **Nombre**: Evaluación de Desempeño
- **Versión**: 1.0

## Objetivo
Evaluar el desempeño del personal periódicamente para identificar fortalezas y áreas de mejora.

## Entradas
- **empleado** (requerido): Nombre del empleado
- **periodo** (requerido): Semestral, anual
- **criterios** (requerido): Competencias a evaluar
- **evidencia** (requerido): Datos objetivos de desempeño

## Proceso
1. Definir criterios de evaluación
2. Recopilar evidencias
3. Aplicar evaluación
4. Calificar competencias
5. Generar retroalimentación
6. Establecer planes de mejora
7. Seguimiento

## Salida
Reporte de evaluación con:
- Calificación por competencia
- Calificación global
- Fortalezas
- Áreas de mejora
- Plan de desarrollo

## Checklist
- [ ] Criterios definidos
- [ ] Evidencias recopiladas
- [ ] Evaluación aplicada
- [ ] Retroalimentación dada
- [ ] Plan de mejora establecido

## Ejemplo
```markdown
# EVALUACIÓN - ING. CARLOS MÉNDEZ
## Periodo: Enero-Junio 2026

## Competencias
| Competencia | Calificación |
|-------------|--------------|
| Liderazgo | 9/10 |
| Técnico | 8/10 |
| Comunicación | 7/10 |
| Puntualidad | 9/10 |

## Global: 8.3/10

## Fortalezas
- Excelente liderazgo de equipo
- Conocimiento técnico sólido

## Mejoras
- Comunicación con clientes
```
