# Skill: analizar_ventas

## Identidad
- **Agente**: Data
- **Nombre**: Análisis de Ventas
- **Versión**: 1.0

## Objetivo
Analizar datos de ventas para identificar tendencias, patrones y oportunidades de mejora.

## Entradas
- **periodo** (requerido): Mes, trimestre, año
- **proyecto** (requerido): Proyecto a analizar
- **tipo_unidad** (requerido): 1 hab, 2 hab, 3 hab
- **canal** (requerido): Referencia, redes, portal

## Proceso
1. Recopilar datos de ventas
2. Limpiar y validar
3. Calcular métricas
4. Identificar tendencias
5. Comparar con períodos anteriores
6. Generar visualizaciones
7. Presentar hallazgos

## Salida
Reporte de análisis con:
- Métricas clave
- Tendencias
- Comparativas
- Recomendaciones

## Checklist
- [ ] Datos recopilados
- [ ] Métricas calculadas
- [ ] Tendencias identificadas
- [ ] Visualizaciones creadas
- [ ] Recomendaciones generadas

## Ejemplo
```markdown
# ANÁLISIS VENTAS - Q2 2026

## Métricas
- Total ventas: $15.2M
- Unidades: 12
- Precio promedio: $1.27M

## Por Canal
| Canal | Unidades | % |
|-------|----------|---|
| Referidos | 5 | 42% |
| Redes | 4 | 33% |
| Portal | 3 | 25% |

## Tendencia
- Crecimiento: +8% vs Q1
- Principal fuente: Referidos
```
