# Skill: generar_dashboard

## Identidad
- **Agente**: Data
- **Nombre**: Generación de Dashboard
- **Versión**: 1.0

## Objetivo
Crear dashboards ejecutivos con indicadores clave para la toma de decisiones.

## Entradas
- **tipo_dashboard** (requerido): Ventas, obras, finanzas, RRHH
- **indicadores** (requerido): KPIs a mostrar
- **audiencia** (requerido): Directivos, gerentes, operativos
- **frecuencia** (requerido): Diario, semanal, mensual

## Proceso
1. Definir indicadores clave
2. Seleccionar fuentes de datos
3. Diseñar layout
4. Crear visualizaciones
5. Automatizar actualización
6. Capacitar usuarios

## Salida
Dashboard con:
- KPIs principales
- Gráficas de tendencia
- Tablas comparativas
- Filtros interactivos

## Checklist
- [ ] KPIs definidos
- [ ] Fuentes identificadas
- [ ] Layout diseñado
- [ ] Visualizaciones creadas
- [ ] Automatización configurada

## Ejemplo
```markdown
# DASHBOARD EJECUTIVO - JULIO 2026

## KPIs
| Indicador | Meta | Actual | Estado |
|-----------|------|--------|--------|
| Ventas | $5M | $4.8M | ⚠️ |
| Avance obra | 75% | 72% | ⚠️ |
| Satisfacción | 9.0 | 9.2 | ✓ |

## Gráficas
- Tendencia ventas mensuales
- Avance por proyecto
- Costos vs presupuesto
```
