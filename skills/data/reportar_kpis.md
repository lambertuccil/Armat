# Skill: reportar_kpis

## Identidad
- **Agente**: Data
- **Nombre**: Reporte de KPIs
- **Versión**: 1.0

## Objetivo
Generar reportes ejecutivos de indicadores clave de desempeño para dirección.

## Entradas
- **periodo** (requerido): Semanal, mensual, trimestral
- **departamento** (requerido): Todos o específico
- **indicadores** (requerido): KPIs a reportar
- **metas** (requerido): Objetivos establecidos

## Proceso
1. Recopilar datos de KPIs
2. Calcular resultados
3. Comparar con metas
4. Analizar desviaciones
5. Generar gráficos
6. Redactar narrativa
7. Distribuir reporte

## Salida
Reporte ejecutivo con:
- Resumen de KPIs
- Comparativa vs metas
- Tendencias
- Acciones correctivas

## Checklist
- [ ] Datos recopilados
- [ ] KPIs calculados
- [ ] Comparativa realizada
- [ ] Desviaciones analizadas
- [ ] Reporte generado

## Ejemplo
```markdown
# REPORTE KPIs - JUNIO 2026

## Resumen
| KPI | Meta | Real | % Cumplimiento |
|-----|------|------|----------------|
| Ventas | $5M | $4.8M | 96% |
| Unidades | 4 | 3 | 75% |
| Satisfacción | 9.0 | 9.2 | 102% |
| Costo/m² | $15K | $14.8K | 101% |

## Análisis
- Ventas: Casi meta alcanzada
- Unidades: Debajo por 1 unidad
- Satisfacción: Superada
- Costos: Controlados

## Acciones
1. Incrementar esfuerzo comercial
2. Revisar pipeline de ventas
```
