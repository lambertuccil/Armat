# Skill: medir_satisfaccion

## Identidad
- **Agente**: Postventa
- **Nombre**: Medición de Satisfacción
- **Versión**: 1.0

## Objetivo
Medir la satisfacción de clientes post-venta para identificar mejoras en el servicio.

## Entradas
- **cliente** (requerido): Cliente a encuestar
- **servicio** (requerido): Servicio prestado
- **momento** (requerido): Post-reparación, post-mantenimiento

## Proceso
1. Diseñar encuesta
2. Enviar al cliente
3. Capturar respuestas
4. Calcular métricas
5. Analizar resultados
6. Implementar mejoras

## Salida
Reporte de satisfacción con:
- NPS
- Calificaciones por aspecto
- Comentarios
- Acciones de mejora

## Checklist
- [ ] Encuesta diseñada
- [ ] Enviada al cliente
- [ ] Respuestas capturadas
- [ ] Métricas calculadas
- [ ] Mejoras implementadas

## Ejemplo
```markdown
# SATISFACCIÓN POST-VENTA

## Cliente: Juan Pérez
## Servicio: Reparación de fuga

## Calificación
- Atención: 9/10
- Tiempo: 10/10
- Calidad: 9/10
- **Global: 9.3/10**

## NPS: 9 (Promotor)

## Comentario
"Excelente servicio, muy rápido."
```
