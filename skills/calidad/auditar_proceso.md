# Skill: auditar_proceso

## Identidad
- **Agente**: Calidad
- **Nombre**: Auditoría de Proceso
- **Versión**: 1.0

## Objetivo
Realizar auditorías de procesos constructivos para verificar cumplimiento de procedimientos y estándares.

## Entradas
- **proceso** (requerido): Proceso a auditar
- **criterios** (requerido): Estándares y procedimientos
- **alcance** (requerido): Departments o áreas a auditar
- **documentacion** (requerido): Procedimientos documentados

## Proceso
1. Planificar auditoría
2. Revisar documentación
3. Observar ejecución
4. Entrevistar personal
5. Verificar evidencias
6. Clasificar hallazgos
7. Generar informe de auditoría

## Salida
Informe de auditoría con:
- Alcance auditado
- Hallazgos por categoría
- Cumplimiento global
- Recomendaciones
- Plan de acción

## Checklist
- [ ] Auditoría planificada
- [ ] Documentación revisada
- [ ] Observaciones realizadas
- [ ] Hallazgos clasificados
- [ ] Informe generado

## Ejemplo
```markdown
# AUDITORÍA - PROCESO DE HORMIGÓN

## Alcance
- Mezclado
- Colado
- Curado

## Hallazgos
| Categoría | Conforme | No Conforme |
|-----------|----------|-------------|
| Mezclado | ✓ | |
| Colado | ✓ | |
| Curado | | ✗ |

## No Conformidad
- Falta registro de temperatura durante curado

## Recomendación
Implementar bitácora de curado
```
