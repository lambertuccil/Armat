# Skill: generar_informe_calidad

## Identidad
- **Agente**: Calidad
- **Nombre**: Informe de Calidad
- **Versión**: 1.0

## Objetivo
Generar informes consolidados de calidad para dirección, mostrando tendencias y cumplimiento.

## Entradas
- **periodo** (requerido): Semanal, mensual
- **inspecciones** (requerido): Resumen de inspecciones
- **no_conformidades** (requerido): Hallazgos encontrados
- **mejoras** (opcional): Acciones implementadas

## Proceso
1. Consolidar datos de inspecciones
2. Calcular índices de calidad
3. Identificar tendencias
4. Analizar causas raíz
5. Proponer mejoras
6. Generar gráficos
7. Redactar informe ejecutivo

## Salida
Informe de calidad con:
- Resumen ejecutivo
- Índices de cumplimiento
- Tendencias
- Acciones correctivas
- Recomendaciones

## Checklist
- [ ] Datos consolidados
- [ ] Índices calculados
- [ ] Tendencias identificadas
- [ ] Informe redactado

## Ejemplo
```markdown
# INFORME DE CALIDAD - JUNIO 2026

## Resumen
- Inspecciones: 24
- Conformes: 21 (87.5%)
- No conformes: 3 (12.5%)

## Principales Hallazgos
1. Acabados superficiales (2 hallazgos)
2. Dimensiones fuera de tolerancia (1 hallazgo)

## Acciones Correctivas
1. Capacitar personal de acabados
2. Revisar procedimiento de medición

## Tendencia
Mejora vs mayo (82% conformidad)
```
