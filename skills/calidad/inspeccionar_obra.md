# Skill: inspeccionar_obra

## Identidad
- **Agente**: Calidad
- **Nombre**: Inspección de Obra
- **Versión**: 1.0

## Objetivo
Realizar inspecciones de calidad en sitio, verificando cumplimiento de especificaciones técnicas y estándares.

## Entradas
- **proyecto** (requerido): Nombre del proyecto
- **area** (requerido): Zona a inspeccionar
- **criterios** (requerido): Estándares a verificar
- **evidencia** (opcional): Fotos, mediciones

## Proceso
1. Definir alcance de inspección
2. Revisar especificaciones técnicas
3. Realizar verificación visual
4. Tomar mediciones
5. Documentar hallazgos
6. Clasificar por severidad
7. Generar reporte de inspección

## Salida
Reporte de inspección con:
- Área inspeccionada
- Hallazgos clasificados
- Mediciones realizadas
- Evidencia fotográfica
- Acciones correctivas

## Checklist
- [ ] Alcance definido
- [ ] Especificaciones revisadas
- [ ] Verificación realizada
- [ ] Hallazgos documentados
- [ ] Reporte generado

## Ejemplo
```markdown
# INSPECCIÓN - NIVEL 3 ENTANYQ

## Áreas Inspeccionadas
- Muros departamentos 301-304
- Instalaciones eléctricas
- Acabados pisos

## Hallazgos
| Hallazgo | Severidad | Acción |
|----------|-----------|--------|
| Grieta muro 302 | Baja | Monitorear |
| cable suelto 303 | Alta | Corregir hoy |

## Evidencia
- 12 fotos tomadas
- 5 mediciones realizadas
```
