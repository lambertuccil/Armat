# Skill: crear_flyer_instagram

## Identidad
- **Nombre**: crear_flyer_instagram
- **Agente**: Marketing
- **Categoría**: Contenido
- **Complejidad**: Media

## Objetivo
Crear un flyer optimizado para publicación en Instagram.

## Entradas
| Parámetro | Tipo | Requerido | Descripción |
|-----------|------|-----------|-------------|
| tema | string | Sí | Tema del flyer (propiedad, avance, promoción) |
| propiedad | string | No | Propiedad relacionada |
| copy | string | No | Copy sugerido |

## Proceso
1. Definir tema y objetivo
2. Seleccionar imagen principal
3. Diseñar estructura (1080x1080)
4. Crear copy atractivo
5. Agregar CTA
6. Incluir hashtags

## Salida
Especificación del flyer con:
- Diseño visual (descripción)
- Copy para publicación
- Hashtags relevantes
- CTA claro

## Checklist
- [ ] Tamaño correcto (1080x1080)
- [ ] Texto mínimo en imagen (regla 20%)
- [ ] Contraste suficiente
- [ ] CTA claro
- [ ] Copy atractivo
- [ ] Hashtags relevantes

## Ejemplo
```
Input: crear_flyer_instagram("avance de obra", "ENTANYQ")

Output:
# Flyer Instagram — Avance de Obra

**Diseño**:
- Fondo: Foto real de obra
- Overlay: Negro 60%
- Texto: "50% — Mampostería completa"

**Copy**:
🏗️ Obra al 50%

Mampostería completa. Próximamente: instalaciones.

📍 Centenario, Neuquén
📞 299 508 9366

#ARMAT #ENTANYQ #Centenario #ObraGris
```
