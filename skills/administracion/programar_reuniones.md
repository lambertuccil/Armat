# Skill: programar_reuniones

## Identidad
- **Agente**: Administración
- **Nombre**: Programación de Reuniones
- **Versión**: 1.0

## Objetivo
Coordinar y programar reuniones de trabajo, juntas de proyecto y sesiones ejecutivas.

## Entradas
- **tipo_reunion** (requerido): Junta proyecto, ejecutiva,ordinaria
- **participantes** (requerido): Lista de asistentes
- **fecha_hora** (requerido): Fecha y hora propuesta
- **duracion** (requerido): Tiempo estimado
- **temas** (requerido): Agenda provisional

## Proceso
1. Verificar disponibilidad de participantes
2. Reservar sala o enlace virtual
3. Elaborar agenda
4. Enviar invitaciones
5. Preparar materiales
6. Documentar acuerdos
7. Dar seguimiento a tareas

## Salida
Invitación a reunión con:
- Fecha, hora, lugar
- Agenda
- Participantes
- Materiales adjuntos

## Checklist
- [ ] Disponibilidad verificada
- [ ] Sala reservada
- [ ] Agenda elaborada
- [ ] Invitaciones enviadas
- [ ] Materiales listos

## Ejemplo
```markdown
# JUNTA DE PROYECTO - ENTANYQ

**Fecha**: 7 julio 2026, 10:00 hrs
**Lugar**: Sala de juntas / Zoom
**Duración**: 1 hora

## Agenda
1. Revisión de avance (15 min)
2. Presupuesto y costos (15 min)
3. Problemas y soluciones (15 min)
4. Próximos pasos (15 min)

## Participantes
- Ing. Carlos Méndez (Obras)
- Lic. Ana García (Legal)
- CPA. María López (Finanzas)
```
