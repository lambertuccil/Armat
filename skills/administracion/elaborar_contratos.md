# Skill: elaborar_contratos

## Identidad
- **Agente**: Administración
- **Nombre**: Elaboración de Contratos Administrativos
- **Versión**: 1.0

## Objetivo
Elaborar borradores de contratos administrativos (suministros, servicios, arrendamientos) para revisión legal.

## Entradas
- **tipo_contrato** (requerido): Suministro, servicio, arrendamiento
- **partes** (requerido): Contratante y contratista
- **objeto** (requerido): Descripción del servicio o suministro
- **condiciones** (requerido): Términos comerciales
- **monto** (opcional): Valor del contrato

## Proceso
1. Definir tipo de contrato
2. Establecer objeto y alcance
3. Definir condiciones comerciales
4. Establecer plazos y entregables
5. Incluir cláusulas de protección
6. Definir mecanismos de resolución
7. Generar borrador

## Salida
Borrador de contrato con:
- Identificación de partes
- Objeto del contrato
- Condiciones comerciales
- Plazos
- Cláusulas estándar

## Checklist
- [ ] Tipo definido
- [ ] Partes identificadas
- [ ] Objeto claro
- [ ] Condiciones establecidas
- [ ] Borrador generado

## Ejemplo
```markdown
# CONTRATO DE SUMINISTRO - BLOQUES

## Partes
- Contratante: ARMAT S.A. de C.V.
- Contratista: Block Norte S.A. de C.V.

## Objeto
Suministro de 5,000 blocks standard para proyecto ENTANYQ

## Condiciones
- Precio: $12 por block
- Entrega: 500 semanales
- Pago: 15 días contra entrega
- Vigencia: 10 semanas
```
