# Skill: proyectar_cierre

## Identidad
- **Agente**: Presupuestos
- **Nombre**: Proyección de Cierre
- **Versión**: 1.0

## Objetivo
Proyectar el cierre financiero de un proyecto, estimando costo final y utilidad.

## Entradas
- **proyecto** (requerido): Nombre del proyecto
- **avance_actual** (requerido): Porcentaje de avance
- **costo_ejecutado** (requerido): Inversión hasta la fecha
- **problemas** (requerido): Issues conocidos

## Proceso
1. Evaluar avance actual
2. Estimar costos pendientes
3. Identificar riesgos y contingencias
4. Calcular costo proyectado
5. Comparar con presupuesto original
6. Estimar utilidad final
7. Generar reporte de cierre

## Salida
Proyección de cierre con:
- Costo proyectado
- Utilidad estimada
- Riesgos identificados
- Recomendaciones

## Checklist
- [ ] Avance evaluado
- [ ] Costos pendientes estimados
- [ ] Riesgos identificados
- [ ] Costo proyectado calculado
- [ ] Reporte generado

## Ejemplo
```markdown
# PROYECCIÓN DE CIERRE - ENTANYQ

## Estado Actual
- Avance: 68%
- Costo ejecutado: $10.5M

## Proyección
- Costo pendiente: $5.2M
- Contingencia: $0.8M
- **Costo final: $16.5M**

## vs Presupuesto
- Original: $15M
- Variación: +$1.5M (+10%)

## Utilidad Esperada
- Ingresos: $22M
- Costos: $16.5M
- Utilidad: $5.5M (25%)
```
