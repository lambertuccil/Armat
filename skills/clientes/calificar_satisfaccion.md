# Skill: calificar_satisfaccion

## Identidad
- **Agente**: Clientes
- **Nombre**: Calificación de Satisfacción
- **Versión**: 1.0

## Objetivo
Medir y evaluar la satisfacción de clientes en diferentes etapas de la experiencia.

## Entradas
- **cliente** (requerido): Nombre del cliente
- **etapa** (requerido): Pre-venta, entrega, post-venta
- **criterios** (requerido): Aspectos a evaluar
- **respuesta** (requerido): Evaluación del cliente

## Proceso
1. Diseñar encuesta de satisfacción
2. Aplicar en momento clave
3. Capturar respuestas
4. Calcular indicadores
5. Identificar áreas de mejora
6. Generar reportes
7. Implementar acciones correctivas

## Salida
Reporte de satisfacción con:
- NPS (Net Promoter Score)
- Calificación por aspecto
- Comentarios destacados
- Acciones de mejora

## Checklist
- [ ] Encuesta diseñada
- [ ] Respuestas capturadas
- [ ] NPS calculado
- [ ] Áreas mejorables identificadas
- [ ] Acciones definidas

## Ejemplo
```markdown
# SATISFACCIÓN - ENTANYQ Q2 2026

## Resultados
- NPS: 72 (Excelente)
- Muestra: 45 clientes

## Por Aspecto
| Aspecto | Calificación |
|---------|--------------|
| Atención | 9.2 |
| Información | 8.8 |
| Proceso compra | 8.5 |
| Entrega | 9.0 |

## Comentarios Destacados
- "Excelente trato personalizado"
- "Proceso claro y transparente"
```
