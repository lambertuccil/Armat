# Skill: buscar_proveedor

## Identidad
- **Agente**: Proveedores
- **Nombre**: Búsqueda de Proveedor
- **Versión**: 1.0

## Objetivo
Buscar y evaluar nuevos proveedores para necesidades específicas de ARMAT.

## Entradas
- **necesidad** (requerido): Material o servicio requerido
- **especificaciones** (requerido): Requisitos técnicos
- **presupuesto** (requerido): Rango de inversión
- **plazo** (requerido): Tiempo de entrega requerido

## Proceso
1. Definir requerimientos
2. Buscar opciones
3. Solicitar cotizaciones
4. Evaluar propuestas
5. Seleccionar proveedor
6. Negociar condiciones
7. Contratar

## Salida
Ficha de nuevo proveedor con:
- Datos generales
- Capacidad
- Precio
- Condiciones

## Checklist
- [ ] Requerimientos definidos
- [ ] Opciones buscadas
- [ ] Cotizaciones recibidas
- [ ] Evaluación realizada
- [ ] Selección documentada

## Ejemplo
```markdown
# BÚSQUEDA: INSTALADOR ELÉCTRICO

## Requerimientos
- Experiencia: 3 años mínimo
- Capacidad: 10 personas
- Disponibilidad: Inmediata

## Opciones Encontradas
1. Eléctricos Unidos (8.2/10)
2. Instalaciones Pro (7.8/10)
3. Soluciones Eléctricas (7.5/10)

## Recomendación: Eléctricos Unidos
- Mejor calificación
- Capacidad suficiente
- Precio competitivo
```
