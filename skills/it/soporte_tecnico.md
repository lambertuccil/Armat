# Skill: soporte_tecnico

## Identidad
- **Agente**: IT
- **Nombre**: Soporte Técnico
- **Versión**: 1.0

## Objetivo
Brindar soporte técnico a usuarios de ARMAT, resolviendo problemas de hardware, software y redes.

## Entradas
- **problema** (requerido): Descripción del problema
- **usuario** (requerido): Usuario afectado
- **equipo** (requerido): Dispositivo involucrado
- **prioridad** (requerido): Alta, media, baja

## Proceso
1. Registrar ticket
2. Clasificar prioridad
3. Diagnosticar problema
4. Solucionar o escalar
5. Documentar solución
6. Cerrar ticket

## Salida
Ticket de soporte con:
- Descripción del problema
- Diagnóstico
- Solución aplicada
- Tiempo de resolución

## Checklist
- [ ] Ticket registrado
- [ ] Prioridad asignada
- [ ] Problema diagnosticado
- [ ] Solución aplicada
- [ ] Documentado

## Ejemplo
```markdown
# TICKET #2026-089

## Problema
- No imprime en color
- Usuario: Ana García
- Equipo: Impresora HP-01
- Prioridad: Media

## Diagnóstico
- Driver desactualizado

## Solución
1. Actualizar driver
2. Configurar predeterminado color

## Tiempo: 30 min
## Estado: CERRADO
```
