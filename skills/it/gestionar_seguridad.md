# Skill: gestionar_seguridad

## Identidad
- **Agente**: IT
- **Nombre**: Gestión de Seguridad Informática
- **Versión**: 1.0

## Objetivo
Proger la infraestructura tecnológica de ARMAT contra amenazas y vulnerabilidades.

## Entradas
- **amenaza** (requerido): Tipo de amenaza
- **nivel_riesgo** (requerido): Alto, medio, bajo
- **accion** (requerido): Prevención, detección, respuesta

## Proceso
1. Identificar amenazas
2. Evaluar vulnerabilidades
3. Implementar controles
4. Monitorear incidentes
5. Responder a amenazas
6. Documentar lecciones

## Salida
Reporte de seguridad con:
- Amenazas detectadas
- Acciones tomadas
- Estado de controles
- Recomendaciones

## Checklist
- [ ] Amenazas identificadas
- [ ] Vulnerabilidades evaluadas
- [ ] Controles implementados
- [ ] Monitoreo activo
- [ ] Incidentes documentados

## Ejemplo
```markdown
# REPORTE SEGURIDAD - JULIO 2026

## Amenazas Detectadas
- Intentos phishing: 5 (bloqueados)
- Virus: 0
- Accesos no autorizados: 0

## Controles
- Firewall: Activo
- Antivirus: Actualizado
- Backups: Completos

## Estado: SEGURO
```
