# Skill: administrar_sistemas

## Identidad
- **Agente**: IT
- **Nombre**: Administración de Sistemas
- **Versión**: 1.0

## Objetivo
Administrar la infraestructura tecnológica de ARMAT, asegurando disponibilidad y seguridad.

## Entradas
- **sistema** (requerido): Sistema a administrar
- **accion** (requerido): Mantenimiento, actualización, monitoreo
- **periodo** (requerido): Diario, semanal, mensual

## Proceso
1. Monitorear disponibilidad
2. Realizar respaldos
3. Aplicar actualizaciones
4. Verificar seguridad
5. Generar reportes

## Salida
Reporte de administración con:
- Estado del sistema
- Respaldos realizados
- Actualizaciones aplicadas
- Incidentes detectados

## Checklist
- [ ] Monitoreo activo
- [ ] Respaldos ejecutados
- [ ] Actualizaciones aplicadas
- [ ] Seguridad verificada
- [ ] Reporte generado

## Ejemplo
```markdown
# REPORTE ADMINISTRACIÓN - JULIO 2026

## Sistemas
| Sistema | Disponibilidad | Estado |
|---------|---------------|--------|
| Google Workspace | 99.9% | OK |
| Servidor archivos | 99.5% | OK |
| Correo | 99.8% | OK |

## Respaldos
- Diarios: 30/30
- Semanales: 4/4
- Mensuales: Pendiente

## Actualizaciones
- Windows: 15 aplicadas
- Office: 8 aplicadas
```
