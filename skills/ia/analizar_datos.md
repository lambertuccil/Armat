# Skill: analizar_datos

## Identidad
- **Agente**: IA
- **Nombre**: Análisis de Datos
- **Versión**: 1.0

## Objetivo
Analizar datos de la empresa para identificar tendencias, patrones y oportunidades de mejora.

## Entradas
- **fuente** (requerido): Ventas, construcción, finanzas, clientes
- **periodo** (requerido): Mes, trimestre, año
- **tipo_analisis** (requerido): Descriptivo, predictivo, prescriptivo
- **variables** (requerido): Variables a analizar

## Proceso
1. Recopilar datos de la fuente
2. Limpiar y estandarizar
3. Aplicar técnicas de análisis
4. Identificar patrones y tendencias
5. Generar visualizaciones
6. Interpretar resultados
7. Recomendar acciones

## Salida
Reporte de análisis con:
- Resumen ejecutivo
- Tendencias identificadas
- Visualizaciones
- Recomendaciones accionables

## Checklist
- [ ] Datos recopilados
- [ ] Datos limpios
- [ ] Análisis realizado
- [ ] Visualizaciones generadas
- [ ] Recomendaciones documentadas

## Ejemplo
```markdown
# ANÁLISIS DE VENTAS - Q2 2026

## Tendencias
- Crecimiento mensual: 8%
- Departamentos 2 hab: mayor demanda
- Canal principal: referidos (45%)

## Predicción Q3
- Ventas esperadas: $28M
- Unidades: 22

## Recomendaciones
1. Invertir en programa de referidos
2. Incrementar inventario 2 habitaciones
```
