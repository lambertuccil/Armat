# Skill: optimizar_procesos

## Identidad
- **Agente**: IA
- **Nombre**: Optimización de Procesos
- **Versión**: 1.0

## Objetivo
Analizar y optimizar procesos de negocio para reducir tiempos, costos y mejorar calidad.

## Entradas
- **proceso** (requerido): Proceso a optimizar
- **metricas_actuales** (requerido): Desempeño actual
- **objetivos** (requerido): Metas de mejora
- **restricciones** (opcional): Limitaciones

## Proceso
1. Mapear proceso actual
2. Identificar cuellos de botella
3. Analizar causas raíz
4. Diseñar proceso optimizado
5. Implementar cambios
6. Medir resultados
7. Estandarizar mejoras

## Salida
Proceso optimizado con:
- Mapa del proceso
- Mejoras implementadas
- Ahorro logrado
- Métricas de seguimiento

## Checklist
- [ ] Proceso mapeado
- [ ] Cuellos de botella identificados
- [ ] Diseño optimizado
- [ ] Cambios implementados
- [ ] Resultados medidos

## Ejemplo
```markdown
# OPTIMIZACIÓN - PROCESO DE COTIZACIÓN

## Actual
- Tiempo: 4 horas por cotización
- Errores: 15%
-客户 satisfacción: 7/10

## Optimizado
- Tiempo: 1.5 horas por cotización
- Errores: 3%
-客户 satisfacción: 9/10

## Ahorro: 62% tiempo, 80% errores
```
