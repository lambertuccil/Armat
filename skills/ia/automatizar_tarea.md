# Skill: automatizar_tarea

## Identidad
- **Agente**: IA
- **Nombre**: Automatización de Tareas
- **Versión**: 1.0

## Objetivo
Identificar y automatizar tareas repetitivas para mejorar eficiencia operativa.

## Entradas
- **tarea** (requerido): Tarea a automatizar
- **frecuencia** (requerido): Diaria, semanal, mensual
- **complejidad** (requerido): Baja, media, alta
- **datos_entrada** (requerido): Inputs necesarios

## Proceso
1. Identificar tarea repetitiva
2. Documentar pasos actuales
3. Diseñar flujo automatizado
4. Implementar automatización
5. Probar y validar
6. Monitorear desempeño
7. Optimizar continuamente

## Salida
Tarea automatizada con:
- Flujo documentado
- Ahorro de tiempo estimado
- Reducción de errores
- Métricas de monitoreo

## Checklist
- [ ] Tarea identificada
- [ ] Pasos documentados
- [ ] Flujo diseñado
- [ ] Implementación realizada
- [ ] Pruebas exitosas

## Ejemplo
```markdown
# AUTOMATIZACIÓN - ENVÍO DE REPORTES

## Tarea Actual
- Preparar reporte manualmente: 2 horas
- Enviar por email: 30 min
- **Total: 2.5 horas/semana**

## Automatización
- Generar reporte automático: 5 min
- Envío programado: automático
- **Total: 5 min/semana**

## Ahorro: 2 horas/semana (87%)
```
