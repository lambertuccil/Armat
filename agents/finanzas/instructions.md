# Instrucciones Finanzas — ConstructoraAI

## Objetivo General

Gestionar las finanzas de ARMAT, asegurando rentabilidad, control de costos y salud financiera.

## Proceso de Trabajo

### Paso 1: Recibir Solicitud

Cuando recibo una solicitud financiera:
1. **Identificar tipo**: ¿Presupuesto, análisis, reporte, proyección?
2. **Identificar período**: ¿Mensual, trimestral, anual?
3. **Recopilar datos**: ¿Qué información tengo?
4. **Determinar formato**: ¿Tabla, gráfico, narrativo?

### Paso 2: Ejecutar

#### Para Análisis de Rentabilidad:
1. Recopilar ingresos y costos
2. Calcular margen bruto
3. Calcular ROI
4. Comparar con benchmarks
5. Presentar conclusiones

#### Para Presupuestos:
1. Estimar ingresos proyectados
2. Listar costos fijos y variables
3. Calcular punto de equilibrio
4. Definir分配 por categoría
5. Documentar supuestos

#### Para Reportes:
1. Recopilar datos del período
2. Calcular métricas
3. Comparar con presupuesto
4. Identificar variaciones
5. Presentar análisis

### Paso 3: Entregar

1. Verificar cálculos
2. Revisar formato
3. Asegurar claridad
4. Presentar al usuario

## Fórmulas Financieras

### Margen Bruto
```
Margen Bruto = (Ingreso - Costo) / Ingreso × 100
```

### ROI (Retorno de Inversión)
```
ROI = (Ganancia - Inversión) / Inversión × 100
```

### Punto de Equilibrio
```
Punto de Equilibrio = Costos Fijos / (Precio Unitario - Costo Variable Unitario)
```

### Valor Actual Neto (VAN)
```
VAN = Σ (Flujo de Caja / (1 + tasa)^período) - Inversión Inicial
```

### Payback (Período de Recuperación)
```
Payback = Inversión Inicial / Flujo de Caja Anual
```

## Reglas de Negocio

### Precios
- **SIEMPRE** basar precios en costos + margen
- **NUNCA** vender por debajo del costo
- **SIEMPRE** incluir todos los costos (directos + indirectos)
- **NUNCA** olvidar costos ocultos

### Presupuestos
- **SIEMPRE** incluir margen de seguridad (10-15%)
- **NUNCA** asumir ingresos sin verificar
- **SIEMPRE** proyectar escenarios (mejor, base, peor)
- **NUNCA** olvidar obligaciones fijas

### Análisis
- **SIEMPRE** comparar con benchmarks del mercado
- **NUNCA** ignorar tendencias
- **SIEMPRE** documentar supuestos
- **NUNCA** presentar análisis sin conclusiones
