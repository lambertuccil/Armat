# Checklist Finanzas — Pre-entrega

## Para Cada Análisis Financiero

### Antes de generar:
- [ ] ¿Identifiqué el tipo de análisis?
- [ ] ¿Tengo datos actualizados?
- [ ] ¿Conozco el período a analizar?

### Durante la generación:
- [ ] ¿Usé fórmulas correctas?
- [ ] ¿Calculé métricas precisas?
- [ ] ¿Comparé con benchmarks?
- [ ] ¿Incluí conclusiones?

### Después de generar:
- [ ] ¿Revisé cálculos?
- [ ] ¿Verifiqué que los datos son correctos?
- [ ] ¿El formato es claro?
- [ ] ¿Está completo?

## Para Cada Presupuesto

### Datos de entrada:
- [ ] Ingresos proyectados realistas
- [ ] Costos fijos identificados
- [ ] Costos variables estimados
- [ ] Supuestos documentados

### Cálculos:
- [ ] Punto de equilibrio calculado
- [ ] Flujo de caja proyectado
- [ ] Margen de seguridad incluido (10-15%)
- [ ] Escenarios proyectados (mejor, base, peor)

### Salida:
- [ ] Desglose claro
- [ ] Métricas incluidas
- [ ] Conclusiones presentes
- [ ] Recomendaciones claras

## Para Cada Reporte

### Datos:
- [ ] Período definido
- [ ] Datos completos
- [ ] Comparación con presupuesto

### Análisis:
- [ ] Variaciones identificadas
- [ ] Causas analizadas
- [ ] Tendencias detectadas
- [ ] Riesgos señalados

### Presentación:
- [ ] Resumen ejecutivo
- [ ] Detalle completo
- [ ] Gráficos si aplica
- [ ] Recomendaciones

## Para Cada Proyección

### Supuestos:
- [ ] Supuestos claros
- [ ] Supuestos realistas
- [ ] Sensibilidad analizada
- [ ] Riesgos identificados

### Cálculos:
- [ ] Flujos de caja proyectados
- [ ] VAN calculado
- [ ] Payback calculado
- [ ] ROI estimado

### Presentación:
- [ ] Escenarios incluidos
- [ ] Comparativa presentada
- [ ] Conclusiones claras
- [ ] Recomendaciones accionables

## Métricas de Calidad

### Para cada análisis:
- [ ] Cálculos correctos (100%)
- [ ] Datos verificados
- [ ] Fórmulas aplicadas correctamente
- [ ] Conclusiones respaldadas
- [ ] Formato profesional
