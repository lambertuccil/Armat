## Ejemplo 6: Análisis de Punto de Equilibrio

### Input del Usuario:
"¿Cuántas unidades tengo que vender para cubrir costos?"

### Análisis:
- **Tipo**: Punto de equilibrio
- **Formato**: Cálculo y análisis

### Output:
```
# Punto de Equilibrio — ENTANYQ

## Datos

### Costos Fijos (Mensuales)
| Categoría | Monto |
|-----------|-------|
| Administración | USD 4.800 |
| Marketing | USD 1.500 |
| **Total Fijos** | **USD 6.300** |

### Costos Variables por Unidad
| Unidad | Costo | Precio | Margen |
|--------|-------|--------|--------|
| 1 Dorm | USD 48.000 | USD 85.000 | USD 37.000 |
| Mono | USD 35.000 | USD 60.000 | USD 25.000 |

## Cálculo Punto de Equilibrio

### Fórmula
```
Unidades = Costos Fijos / Margen Unitario
```

### Para 1 Dormitorio
```
Unidades = USD 6.300 / USD 37.000 = 0.17 unidades/mes
```

### Para Monoambiente
```
Unidades = USD 6.300 / USD 25.000 = 0.25 unidades/mes
```

### Mix de Ventas (50% cada uno)
```
Margen promedio = (37.000 + 25.000) / 2 = USD 31.000
Unidades = USD 6.300 / USD 31.000 = 0.20 unidades/mes
```

## Análisis

### Para Cubrir Costos Fijos:
- **Solo 1 dorm**: 0.17 unidades/mes (1 cada 6 meses)
- **Solo mono**: 0.25 unidades/mes (1 cada 4 meses)
- **Mix**: 0.20 unidades/mes (1 cada 5 meses)

### Situación Actual
- **Ventas proyectadas**: 1 unidad/mes (mix)
- **Punto de equilibrio**: 0.20 unidades/mes
- **Margen de seguridad**: 80%

## Conclusión
- **Muy por encima del punto de equilibrio**
- **Margen de seguridad amplio**
- **Riesgo de pérdida bajo**

## Recomendaciones
1. Mantener ritmo de ventas actual
2. No bajar de 1 unidad/mes
3. Controlar costos fijos
4. Optimizar margen por unidad
```
