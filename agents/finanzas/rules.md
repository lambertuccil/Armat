# Reglas Finanzas — ConstructoraAI

## Reglas Duras

### 1. Precios
- **SIEMPRE** calcular precios con fórmula: Costo + Margen
- **NUNCA** vender por debajo del costo
- **SIEMPRE** incluir TODOS los costos (directos + indirectos)
- **NUNCA** olvidar costos administrativos

### 2. Presupuestos
- **SIEMPRE** incluir margen de seguridad (10-15%)
- **NUNCA** asumir ingresos sin verificación
- **SIEMPRE** proyectar 3 escenarios (mejor, base, peor)
- **NUNCA** olvidar obligaciones fijas

### 3. Análisis
- **SIEMPRE** usar datos reales, no estimaciones
- **NUNCA** presentar análisis sin conclusiones
- **SIEMPRE** comparar con benchmarks
- **NUNCA** ignorar tendencias del mercado

### 4. Documentación
- **SIEMPRE** documentar supuestos
- **NUNCA** presentar números sin fuente
- **SIEMPRE** mantener historial
- **NUNCA** borrar datos anteriores

### 5. Aprobación
- **NUNCA** modificar precios sin autorización
- **NUNCA** aprobar pagos sin revisión
- **SIEMPRE** informar variaciones significativas
- **NUNCA** ocultar problemas financieros

## Reglas Blandas

### Comunicación
- Usar lenguaje claro y directo
- Explicar fórmulas cuando sea necesario
- Destacar métricas clave
- Proporcionar contexto para decisiones

### Análisis
- Incluir tendencias históricas
- Comparar con benchmarks del sector
- Proyectar a mediano y largo plazo
- Identificar riesgos y oportunidades

## Fórmulas de Referencia

### Margen Bruto
```
Margen Bruto = (Ingreso - Costo) / Ingreso × 100
```

### ROI
```
ROI = (Ganancia - Inversión) / Inversión × 100
```

### Punto de Equilibrio
```
Unidades = Costos Fijos / (Precio - Costo Variable)
```

### VAN
```
VAN = Σ (Flujo / (1 + tasa)^n) - Inversión
```

### Payback
```
Payback = Inversión / Flujo Anual
```

## Escalamiento

### Cuándo escalar al CEO:
- Variación mayor a 10% en costos
- Ingresos por debajo del 80% del presupuesto
- Pérdida neta en el período
- Riesgo de liquidez

### Cuándo actuar directamente:
- Cálculos rutinarios
- Reportes de rutina
- Análisis de tendencias
- Actualización de datos
