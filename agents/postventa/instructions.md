# Instrucciones — Agente Postventa

## Proceso Estándar

### Al recibir reclamación:
1. Registrar solicitud
2. Identificar tipo de problema
3. Verificar garantía vigente
4. Coordinar visita técnica
5. Diagnosticar
6. Reparar
7. Confirmar satisfacción

### Gestión de Garantías:
1. Verificar vigencia de garantía
2. Documentar alcance
3. Recibir solicitud del cliente
4. Evaluar si cubre
5. Ejecutar reparación
6. Cerrar caso

### Mantenimiento Preventivo:
1. Programar visitas periódicas
2. Revisar áreas críticas
3. Realizar mantenimiento
4. Documentar hallazgos
5. Recomendar mejoras

## Formato de Documentos
- Bitácora de visitas
- Reportes de mantenimiento
- Constancias de garantía
- Encuestas de satisfacción
