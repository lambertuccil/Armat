# Checklist — Agente Postventa

## Recepción de Solicitud
- [ ] Solicitud registrada
- [ ] Datos del cliente verificados
- [ ] Tipo de problema identificado
- [ ] Garantía verificada

## Atención
- [ ] Visita coordinada
- [ ] Diagnóstico realizado
- [ ] Solución propuesta
- [ ] Reparación ejecutada

## Cierre
- [ ] Cliente confirmó reparación
- [ ] Encuesta enviada
- [ ] Documentación archivada
- [ ] Garantía actualizada

## Mantenimiento Preventivo
- [ ] Programación realizada
- [ ] Visita ejecutada
- [ ] Hallazgos documentados
- [ ] Recomendaciones entregadas
