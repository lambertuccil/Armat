# Checklist Comercial — Pre-entrega

## Para Cada Propuesta de Venta

### Antes de generar:
- [ ] ¿Identifiqué la unidad correctamente?
- [ ] ¿Verifiqué disponibilidad?
- [ ] ¿Tengo datos del cliente?
- [ ] ¿Conozco sus necesidades?

### Durante la generación:
- [ ] ¿Usé precios de la tabla oficial?
- [ ] ¿Calculé USD/m² correctamente?
- [ ] ¿Incluí opciones contado y financiado?
- [ ] ¿Agregué beneficios de la zona?
- [ ] ¿Incluí disclaimer de renders?

### Después de generar:
- [ ] ¿Revisé errores ortográficos?
- [ ] ¿Verifiqué que los precios son correctos?
- [ ] ¿El formato es profesional?
- [ ] ¿Está completo?

## Para Cada Cotización

### Datos de entrada:
- [ ] Tipo de unidad identificada
- [ ] Superficie confirmada
- [ ] Precio official verificado
- [ ] Financiamiento calculado

### Datos de salida:
- [ ] Precio contado presentado
- [ ] USD/m² incluido
- [ ] Cuotas calculadas correctamente
- [ ] Condiciones claras

## Para Cada Seguimiento

### Registro:
- [ ] Fecha de interacción registrada
- [ ] Nombre del contacto guardado
- [ ] Motivo de consulta documentado
- [ ] Próximo paso definido

### Acción:
- [ ] Follow-up programado
- [ ] Propuesta enviada (si aplica)
- [ ] Estado actualizado en pipeline
- [ ] Notas agregadas al perfil

## Para Cada Negociación

### Análisis:
- [ ] Necesidades reales del cliente identificadas
- [ ] Presupuesto confirmado
- [ ] Urgencia evaluada
- [ ] Alternativas presentadas

### Propuesta:
- [ ] Beneficios enfatizados
- [ ] Objeciones respondidas
- [ ] Valor justificado
- [ ] Próximo paso claro

### Cierre:
- [ ] Condiciones acordadas
- [ ] Documentación preparada
- [ ] Fecha de cierre estimada
- [ ] Follow-up programado

## Para Cada Error

### Detección:
- [ ] ¿Identifiqué el error?
- [ ] ¿Es un error de precios?
- [ ] ¿Es un error de formato?
- [ ] ¿Es un error de información?

### Corrección:
- [ ] Corregí el error
- [ ] Informé al usuario
- [ ] Actualicé el documento
- [ ] Verifiqué que no se repite

## Métricas de Calidad

### Para cada propuesta:
- [ ] Precios correctos (100%)
- [ ] USD/m² calculado correctamente
- [ ] Formato profesional
- [ ] Sin errores ortográficos
- [ ] Disclaimer incluido
- [ ] Datos de contacto correctos

### Para cada interacción:
- [ ] Tiempo de respuesta < 24h
- [ ] Información completa
- [ ] Tono profesional
- [ ] Próximo paso claro
