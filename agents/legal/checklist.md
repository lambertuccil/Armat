# Checklist Legal — Pre-entrega

## Para Cada Contrato

### Antes de redactar:
- [ ] ¿Identifiqué el tipo de contrato?
- [ ] ¿Tengo todos los datos de las partes?
- [ ] ¿Conozco las condiciones específicas?
- [ ] ¿Verifiqué que no hay irregularidades?

### Durante la redacción:
- [ ] ¿Usé lenguaje claro y preciso?
- [ ] ¿Incluí todas las cláusulas obligatorias?
- [ ] ¿Especificué plazos exactos?
- [ ] ¿Detallé precio y forma de pago?
- [ ] ¿Incluí causales de resolución?

### Después de redactar:
- [ ] ¿Revisé errores legales?
- [ ] ¿Verifiqué que protege a ARMAT?
- [ ] ¿Es claro para ambas partes?
- [ ] ¿Está completo?

## Para Cada Revisión

### Documento:
- [ ] ¿Leí el documento completo?
- [ ] ¿Identifiqué puntos críticos?
- [ ] ¿Señalé riesgos?
- [ ] ¿Propuse correcciones?

### Análisis:
- [ ] ¿Es legalmente válido?
- [ ] ¿Protege a ARMAT?
- [ ] ¿Es justo para ambas partes?
- [ ] ¿Cumple normativa?

### Presentación:
- [ ] ¿El análisis es claro?
- [ ] ¿Incluyo recomendaciones?
- [ ] ¿Proporciono opciones?
- [ ] ¿Documenté la decisión?

## Para Cada Asesoría

### Análisis:
- [ ] ¿Entendí la situación?
- [ ] ¿Identifiqué opciones legales?
- [ ] ¿Evalúé riesgos?
- [ ] ¿Recomendé acción clara?

### Documentación:
- [ ] ¿Documenté la asesoría?
- [ ] ¿Registré la decisión?
- [ ] ¿Archivé correctamente?
- [ ] ¿Informé al usuario?

## Para Cada Permisos

### Verificación:
- [ ] ¿Identifiqué permisos necesarios?
- [ ] ¿Verifiqué vencimientos?
- [ ] ¿Confirmé que están al día?
- [ ] ¿Programé renovaciones?

### Gestión:
- [ ] ¿Solicité con anticipación?
- [ ] ¿Documenté trámites?
- [ ] ¿Archivé comprobantes?
- [ ] ¿Informé al usuario?

## Métricas de Calidad

### Para cada contrato:
- [ ] Lenguaje claro (100%)
- [ ] Cláusulas completas
- [ ] Plazos exactos
- [ ] Protección ARMAT
- [ ] Sin ambigüedades

### Para cada revisión:
- [ ] Puntos críticos identificados
- [ ] Riesgos señalados
- [ ] Correcciones propuestas
- [ ] Recomendaciones claras
