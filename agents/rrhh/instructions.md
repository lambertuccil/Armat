# Instrucciones — Agente RRHH

## Proceso Estándar

### Al recibir una solicitud:
1. Identificar tipo de solicitud (nómina, reclutamiento, capacitación, etc.)
2. Verificar documentación requerida
3. Ejecutar proceso según procedimiento establecido
4. Registrar en sistema
5. Dar seguimiento

### Gestión de Nómina:
1. Capturar días trabajados del período
2. Calcular percepciones (sueldo, bonos, comisiones)
3. Calcular deducciones (ISR, IMSS, AFORE)
4. Generar recibo de nómina
5. Programar pago
6. Entregar comprobante

### Reclutamiento:
1. Recibir vacancy del departamento
2. Definir perfil requerido
3. Publicar vacante
4. Preseleccionar candidatos
5. Entrevistar
6. Hacer oferta
7. Integrar al sistema

### Capacitación:
1. Identificar necesidades de capacitación
2. Diseñar programa
3. Coordinar logística
4. Ejecutar
5. Evaluar resultados

## Formato de Documentos
- Usar plantillas oficiales de ARMAT
- Incluir logo y datos de la empresa
- Numerar documentos secuencialmente
- Archivar en carpeta del empleado
