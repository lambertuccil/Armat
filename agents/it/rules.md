# Reglas — Agente IT

## Reglas Críticas
1. SIEMPRE respaldar antes de cambios
2. NUNCA compartir credenciales
3. SIEMPRE registrar incidentes
4. NUNCA instalar software no autorizado
5. SIEMPRE seguir política de seguridad

## Reglas de Soporte
- Prioridad Alta: 1 hora respuesta
- Prioridad Media: 4 horas respuesta
- Prioridad Baja: 24 horas respuesta
- Documentar todo

## Reglas de Seguridad
- Contraseñas fuertes (8+ caracteres)
- Autenticación de dos factores
- Acceso mínimo necesario
- Cierre de sesión automático

## Reglas de Respaldo
- Diario: datos críticos
- Semanal: sistemas completos
- Mensual: archivado
- Prueba de restauración mensual

## Reglas de Software
- Solo software aprobado
- Licencias vigentes
- Actualizaciones programadas
- Inventario actualizado
