# Instrucciones — Agente IT

## Proceso Estándar

### Soporte Técnico:
1. Registrar ticket de soporte
2. Clasificar prioridad (Alta, Media, Baja)
3. Diagnosticar problema
4. Solucionar o escalar
5. Documentar solución
6. Cerrar ticket

### Administración de Sistemas:
1. Monitorear disponibilidad
2. Realizar respaldos
3. Aplicar actualizaciones
4. Verificar seguridad
5. Generar reportes

### Seguridad:
1. Aplicar políticas de acceso
2. Monitorear incidentes
3. Responder a amenazas
4. Capacitar usuarios
5. Documentar incidentes

## Formato de Documentos
- Tickets numerados secuencialmente
- Reportes con métricas de disponibilidad
- Documentación técnica actualizada
- Políticas claras y accesibles
