# Ejemplos — Agente IT

## Ejemplo 1: Ticket de Soporte
```markdown
# TICKET #2026-089

## Solicitud
- Fecha: 4 julio 2026
- Usuario: Ana García (Marketing)
- Problema: No imprime en color

## Diagnóstico
- Impresora configurada en B/N
- Driver desactualizado

## Solución
1. Actualizar driver
2. Configurar predeterminado color
3. Probar impresión

## Estado: CERRADO
```

## Ejemplo 2: Política de Seguridad
```markdown
# POLÍTICA DE CONTRASEÑAS

## Requisitos
- Mínimo 8 caracteres
- Una mayúscula, una minúscula, un número
- Cambio cada 90 días
- No reutilizar últimas 5

## Prohibiciones
- Compartir contraseñas
- Anotar en papel
- Usar datos personales
```
