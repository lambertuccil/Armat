1. **Renders para vender**: Usar renders para mostrar el proyecto final (venden el sueño)
2. **Fotos reales para estado**: Solo en sección "Estado de Obra" (honestidad)
3. **Paleta premium**: Negro #000 + Dorado #C9A96E (imagen lujosa)
4. **Sin look genérico IA**: No cream+serif+terracotta, no negro+verde ácido
5. **Optimización WhatsApp**: Formato 750px ancho para envío por mensajería

6. **USD/m² siempre visible**: Para comparación transparente
7. **Opciones de pago**: Siempre mostrar contado vs financiado
8. **Disclaimer obligatorio**: En cada pieza con renders
9. **Tabla de precios oficial**: No inventar, usar siempre la tabla

### Técnico
10. **Formato HTML single-file**: Brochures autocontenidos (CSS inline)
11. **Sin superposiciones**: Layout limpio y profesional

## Lecciones Aprendidas

### Lo que funciona:
- Renders de fachada venden mucho
- USD/m² genera confianza en el cliente
- Disclaimer claro evita problemas legales
- Formato WhatsApp es el más usado para envíos

### Lo que NO funciona:
- Demasiados overlays en el diseño → se ve amateur
- Sin disclaimer en renders → problema legal
- Precios sin USD/m² → cliente no puede comparar
- Look genérico de IA → no genera confianza

### Errores a evitar:
- Usar colores fuera de paleta
- Superponer elementos sin necesidad
- No incluir datos de contacto
- Olvidar disclaimer de renders
