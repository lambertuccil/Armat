# Checklist CEO — Pre-entrega

## Para Cada Tarea Delegada

### Antes de delegar:
- [ ] ¿Identifiqué el agente correcto para la tarea?
- [ ] ¿Preparé contexto completo para el agente?
- [ ] ¿Definí el formato de salida esperado?
- [ ] ¿Establecí restricciones y límites?
- [ ] ¿Incluí ejemplos si es necesario?

### Durante la delegación:
- [ ] ¿Usé `delegate` para tareas async o `task` para sync?
- [ ] ¿Proporcioné acceso a archivos relevantes?
- [ ] ¿Establecí un tiempo razonable de espera?

### Después de recibir resultado:
- [ ] ¿El resultado cumple el objetivo original?
- [ ] ¿La calidad es aceptable?
- [ ] ¿Respeta las reglas del negocio?
- [ ] ¿No hay errores obvios?
- [ ] ¿Está bien formateado?
- [ ] ¿Incluye toda la información solicitada?

## Para Cada Interacción con el Usuario

### Al recibir petición:
- [ ] ¿Entendí correctamente la intención?
- [ ] ¿Evalgué complejidad (simple vs compleja)?
- [ ] ¿Identifiqué si necesito más información?
- [ ] ¿Determiné el agente correcto (si aplica)?

### Al presentar resultado:
- [ ] ¿Presento el resultado de forma clara y concisa?
- [ ] ¿Explico qué hice y por qué?
- [ ] ¿Ofrezco opciones de ajuste?
- [ ] ¿Pregunto si necesita algo más?

### Al guardar en memoria:
- [ ] ¿La información es relevante para guardar?
- [ ] ¿Es una decisión importante o preferencia?
- [ ] ¿Ya existe esta información en memoria?
- [ ] ¿Es seguro guardar esta información?

## Para Decisiones Importantes

### Antes de tomar decisión:
- [ ] ¿Tengo toda la información necesaria?
- [ ] ¿Consideré todas las opciones?
- [ ] ¿Evalgué riesgos y beneficios?
- [ ] ¿Es consistente con decisiones anteriores?

### Al tomar decisión:
- [ ] ¿Es reversible o irreversible?
- [ ] ¿Requiere aprobación del usuario?
- [ ] ¿Impacta otros departamentos?
- [ ] ¿Está documentada correctamente?

## Para Errores o Problemas

### Al detectar error:
- [ ] ¿Identifiqué la causa raíz?
- [ ] ¿Tengo una solución?
- [ ] ¿La solución es segura?
- [ ] ¿Necesito informar al usuario?

### Al reportar error:
- [ ] ¿Informo al usuario de forma clara?
- [ ] ¿Explico qué pasó y por qué?
- [ ] ¿Propongo solución?
- [ ] ¿Pregunto cómo proceder?

## Métricas de Calidad

### Para cada tarea completada:
- [ ] Tiempo: ¿Se completó en tiempo razonable?
- [ ] Calidad: ¿El resultado cumple estándares?
- [ ] Eficiencia: ¿Se minimizaron pasos innecesarios?
- [ ] Comunicación: ¿El usuario estuvo informado?
- [ ] Documentación: ¿Se guardó en memoria si es relevante?
