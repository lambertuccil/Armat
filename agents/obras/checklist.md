# Checklist Obras — Pre-entrega

## Para Cada Reporte de Avance

### Antes de generar:
- [ ] ¿Identifiqué el proyecto correctamente?
- [ ] ¿Tengo datos actualizados?
- [ ] ¿Conozco el estado de cada etapa?

### Durante la generación:
- [ ] ¿Calculé porcentajes correctamente?
- [ ] ¿Incluí todas las etapas?
- [ ] ¿Documenté observaciones?
- [ ] ¿Identifiqué próximos pasos?

### Después de generar:
- [ ] ¿Revisé errores de cálculo?
- [ ] ¿Verifiqué que los datos son correctos?
- [ ] ¿El formato es claro?
- [ ] ¿Está completo?

## Para Cada Cálculo de Materiales

### Datos de entrada:
- [ ] Superficie confirmada
- [ ] Especificaciones claras
- [ ] Precios actualizados

### Cálculos:
- [ ] Cantidades correctas
- [ ] Costos estimados realistas
- [ ] Incluí margen de desperdicio
- [ ] Total verificado

### Salida:
- [ ] Desglose claro
- [ ] Costo total incluido
- [ ] Notas si es necesario

## Para Cada Coordinación

### Planificación:
- [ ] Tareas identificadas
- [ ] Tiempos estimados
- [ ] Secuencia definida
- [ ] Dependencias claras

### Ejecución:
- [ ] Comunicación con subcontratistas
- [ ] Seguimiento de avance
- [ ] Documentación de cambios
- [ ] Reporte de progreso

### Control:
- [ ] Avance vs plan comparado
- [ ] Retrasos identificados
- [ ] Soluciones propuestas
- [ ] Actualización de cronograma

## Para Cada Problema

### Detección:
- [ ] ¿Identifiqué el problema?
- [ ] ¿Es de calidad, seguridad o cronograma?
- [ ] ¿Cuál es el impacto?
- [ ] ¿Tengo solución?

### Acción:
- [ ] Documenté el problema
- [ ] Informé al responsable
- [ ] Propuse solución
- [ ] Programé seguimiento

### Cierre:
- [ ] Problema resuelto
- [ ] Documentación actualizada
- [ ] Lecciones aprendidas
- [ ] Prevención para futuro

## Métricas de Calidad

### Para cada reporte:
- [ ] Datos correctos (100%)
- [ ] Cálculos verificados
- [ ] Formato claro
- [ ] Observaciones documentadas
- [ ] Próximos pasos definidos

### Para cada cálculo:
- [ ] Cantidades precisas
- [ ] Costos realistas
- [ ] Desglose claro
- [ ] Total verificado
