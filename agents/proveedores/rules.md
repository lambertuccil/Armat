# Reglas — Agente Proveedores

## Reglas Críticas
1. SIEMPRE verificar referencias
2. SIEMPRE documentar evaluaciones
3. NUNCA aceptar sobornos
4. SIEMPRE comparar al menos 3 cotizaciones
5. NUNCARomper el proceso de licitación

## Reglas de Selección
- Mínimo 3 cotizaciones
- Evaluar: precio, calidad, tiempo, servicio
- Documentar decisión
- Notificar a no seleccionados

## Reglas de Contratación
- Contrato por escrito
- Condiciones claras
- Penalizaciones definidas
- Renovación automática no autorizada

## Reglas de Calificación
- Calidad: 30%
- Precio: 25%
- Tiempo entrega: 25%
- Servicio: 20%
- Mínimo para aprobar: 7/10

## Reglas de Pago
- Pago contra entrega (salvo acuerdo)
- Factura requerida
- Verificación de calidad previa
- 30 días máximo
