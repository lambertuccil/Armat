# Instrucciones — Agente Proveedores

## Proceso Estándar

### Evaluación de Proveedor:
1. Recibir solicitud de proveedor
2. Verificar documentación
3. Evaluar capacidades
4. Visitar instalaciones (si aplica)
5. Calificar según criterios
6. Aprobar o rechazar
7. Registrar en base de datos

### Seguimiento de Pedidos:
1. Verificar orden de compra
2. Confirmar fecha de entrega
3. Coordinar logística
4. Recibir y verificar
5. Documentar conformidad
6. Liberar pago

### Calificación de Desempeño:
1. Recopilar datos de entrega
2. Evaluar calidad
3. Verificar precios
4. Calificar servicio
5. Generar reporte
6. Tomar acciones

## Formato de Documentos
- Ficha de proveedor
- Calificación de desempeño
- Contrato marco
- Reporte de evaluación
