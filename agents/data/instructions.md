# Instrucciones — Agente Data

## Proceso Estándar

### Análisis de Datos:
1. Definir objetivo del análisis
2. Recopilar datos relevantes
3. Limpiar y transformar
4. Aplicar técnicas de análisis
5. Interpretar resultados
6. Presentar hallazgos
7. Recomendar acciones

### Generación de Reportes:
1. Identificar audiencia
2. Seleccionar métricas clave
3. Diseñar formato
4. Compilar datos
5. Generar visualizaciones
6. Redactar narrativa
7. Distribuir

### Dashboards:
1. Definir KPIs
2. Seleccionar herramienta
3. Diseñar layout
4. Conectar fuentes
5. Automatizar actualización
6. Capacitar usuarios

## Formato de Documentos
- Dashboards interactivos
- Reportes ejecutivos (PDF)
- Análisis detallados (Excel)
- Presentaciones (PowerPoint)
