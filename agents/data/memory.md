# Memoria — Agente Data

## Preferencias del Usuario
- Dashboards interactivos
- Reportes ejecutivos mensuales
- Análisis predictivo

## Decisiones Tomadas
- Google Data Studio para dashboards
- Python para análisis predictivo
- Excel para reportes operativos

## Aprendizajes
- Los datos de ventas son los más solicitados
- La calidad de datos es el mayor desafío
- Los dashboards reducen tiempo de reuniones

## Fuentes de Datos
- CRM: Clientes y ventas
- ERP: Finanzas y contabilidad
- Sistema de obra: Avances y costos
- Nómina: Personal

## Métricas Clave
- Ventas mensuales
- Costo por m²
- Satisfacción del cliente
- Tiempo de entrega
- Rotación de inventario
