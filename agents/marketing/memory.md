## Decisiones de Diseño

### Formatos Aprobados
1. **Brochure HTML**: Single-file, CSS inline, 750px WhatsApp
2. **Flyer Instagram**: 1080x1080px, minimalista
3. **Email Marketing**: HTML responsive, 600px max

### Reglas Clave
1. **NUNCA** usar look genérico de IA
2. **SIEMPRE** usar paleta ARMAT
3. **SIEMPRE** incluir disclaimer en renders
4. **SIEMPRE** mostrar USD/m²
5. **NUNCA** superponer elementos
6. **SIEMPRE** mantener espacio negativo

### Renders vs Fotos Reales
- **Renders**: Para vender el proyecto final (portada, interiores)
- **Fotos reales**: Solo para estado de obra (honestidad)

## Lecciones Aprendidas

### Lo que funciona:
- Renders de fachada son impactantes
- USD/m² genera confianza
- Formato WhatsApp es el más usado
- Disclaimer claro evita problemas

### Lo que NO funciona:
- Demasiados overlays → se ve amateur
- Sin disclaimer → problema legal
- Look genérico IA → no genera confianza
- Superposiciones → confunde al usuario

### Errores a evitar:
- Colores fuera de paleta
- Tipografía incorrecta
- Sin datos de contacto
- Olvidar disclaimer
