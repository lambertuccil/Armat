# Instrucciones Marketing — ConstructoraAI

## Objetivo General

Crear material de marketing profesional que posicione ARMAT como marca premium, generando contenido que venda y genere confianza en clientes potenciales.

## Proceso de Trabajo

### Paso 1: Recibir Briefing

Cuando recibo una solicitud de contenido:

1. **Identificar formato**: ¿Qué se necesita? (brochure, flyer, post, email)
2. **Identificar audience**: ¿Para quién es?
3. **Identificar plataforma**: ¿Dónde se va a publicar?
4. **Identificar objetivo**: ¿Qué se quiere lograr?

### Paso 2: Preparar Recursos

1. **Consultar reglas de marca**: Paleta, tipografía, tono
2. **Seleccionar renders/fotos**: ¿Qué imágenes usar?
3. **Recopilar información**: Precios, características, datos
4. **Definir estructura**: Layout y jerarquía

### Paso 3: Crear Contenido

1. **Diseñar estructura**: Wireframe mental
2. **Crear HTML/CSS**: Código limpio y ordenado
3. **Insertar contenido**: Textos, imágenes, datos
4. **Aplicar estilos**: Paleta y tipografía ARMAT
5. **Optimizar**: Responsive, WhatsApp, impresión

### Paso 4: Revisar y Entregar

1. **Auto-revisión**: ¿Cumple reglas de marca?
2. **Revisión de calidad**: ¿Se ve profesional?
3. **Revisión técnica**: ¿Funciona en móvil?
4. **Presentar al usuario**: Opciones y ajustes
5. **Entregar**: Archivo final

## Formatos de Salida

### Brochure HTML (WhatsApp + Impresión)

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[Título]</title>
    <link href="https://fonts.googleapis.com/css2?family=DM+Sans:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600&display=swap" rel="stylesheet">
    <style>
        /* CSS inline aquí */
    </style>
</head>
<body>
    <!-- Contenido aquí -->
</body>
</html>
```

**Características:**
- Single-file (todo junto)
- CSS inline (sin dependencias externas)
- Google Fonts CDN para tipografía
- Optimizado para 750px (WhatsApp)
- Print-ready para A4

### Flyer Instagram

**Especificaciones:**
- Tamaño: 1080x1080px
- Formato: PNG o JPG
- Resolución: 72dpi (web)
- Espacio de seguridad: 100px bordes

**Estructura:**
1. Imagen de impacto (render o foto)
2. Título corto
3. Dato clave (precio o beneficio)
4. CTA (llamado a la acción)
5. Logo/marca

### Email Marketing

**Especificaciones:**
- Ancho máximo: 600px
- Responsive: Siempre
- Imágenes: Inline o CDN
- Botones: 44px mínimo (touch)

**Estructura:**
1. Header con logo
2. Contenido principal
3. CTA principal
4. Información adicional
5. Footer con contacto

## Reglas de Diseño

### Paleta de Colores
- **NUNCA** usar colores fuera de la paleta
- **SIEMPRE** mantener contraste suficiente
- **SIEMPRE** usar dorado como acento, no como color principal
- **NUNCA** usar más de 3 colores por pieza

### Tipografía
- **NUNCA** usar más de 2 fuentes
- **SIEMPRE** mantener jerarquía clara
- **SIEMPRE** usar tamaños legibles (mínimo 11px)
- **NUNCA** usar tipografía decorativa para cuerpo

### Imágenes
- **SIEMPRE** usar imágenes de alta calidad
- **NUNCA** estirar o distorsionar imágenes
- **SIEMPRE** mantener proporciones
- **SIEMPRE** incluir disclaimer en renders

### Layout
- **NUNCA** superponer elementos innecesariamente
- **SIEMPRE** mantener espaciado consistente
- **SIEMPRE** usar grid o flexbox para alinear
- **NUNCA** llenar todo el espacio (espacio negativo)
