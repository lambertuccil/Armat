# Checklist Marketing — Pre-entrega

## Para Cada Pieza de Marketing

### Antes de crear:
- [ ] ¿Identifiqué el formato correcto?
- [ ] ¿Conozco la plataforma de destino?
- [ ] ¿Tengo la información de la propiedad?
- [ ] ¿Seleccioné los renders/fotos?
- [ ] ¿Consulté las reglas de marca?

### Durante la creación:
- [ ] ¿Uso paleta ARMAT?
- [ ] ¿Uso tipografía correcta?
- [ ] ¿Mantengo espacio negativo?
- [ ] ¿Evito superposiciones?
- [ ] ¿Incluyo disclaimer en renders?
- [ ] ¿Muestro USD/m²?
- [ ] ¿Incluyo datos de contacto?

### Después de crear:
- [ ] ¿Revisé errores ortográficos?
- [ ] ¿Verifiqué que se ve bien en móvil?
- [ ] ¿Optimicé para WhatsApp (750px)?
- [ ] ¿Es consistente con otras piezas?
- [ ] ¿Cumple objetivos?

## Para Brochure HTML

### Estructura:
- [ ] HTML5 válido
- [ ] CSS inline (sin archivos externos)
- [ ] Google Fonts CDN incluido
- [ ] Meta viewport configurado
- [ ] Single-file (todo junto)

### Contenido:
- [ ] Portada con render de fachada
- [ ] Página de unidades con renders interiores
- [ ] Página de estado con fotos reales
- [ ] Página de contacto
- [ ] Disclaimer en cada página con renders

### Diseño:
- [ ] Paleta correcta (negro + dorado + blanco)
- [ ] Tipografía correcta (Playfair + DM Sans)
- [ ] Sin superposiciones
- [ ] Jerarquía clara
- [ ] Espaciado consistente

### Técnico:
- [ ] Responsive (750px WhatsApp)
- [ ] Imágenes optimizadas
- [ ] Sin errores de renderizado
- [ ] Funciona en navegador móvil

## Para Flyer Instagram

### Estructura:
- [ ] 1080x1080px
- [ ] Formato PNG o JPG
- [ ] Resolución 72dpi
- [ ] Espacio de seguridad 100px

### Contenido:
- [ ] Imagen de impacto
- [ ] Título corto
- [ ] Dato clave (precio o beneficio)
- [ ] CTA claro
- [ ] Logo/marca

### Diseño:
- [ ] Minimalista
- [ ] Sin texto en imagen (regla 20%)
- [ ] Contraste suficiente
- [ ] Legible en tamaño pequeño

## Para Email Marketing

### Estructura:
- [ ] Ancho máximo 600px
- [ ] Responsive
- [ ] Imágenes inline o CDN
- [ ] Botones 44px mínimo

### Contenido:
- [ ] Header con logo
- [ ] Contenido principal
- [ ] CTA principal
- [ ] Información adicional
- [ ] Footer con contacto

### Diseño:
- [ ] Limpio y profesional
- [ ] Sin adornos innecesarios
- [ ] Jerarquía clara
- [ ] Espaciado adecuado

## Para Cada Publicación

### Copy:
- [ ] Tono premium
- [ ] Extensión adecuada
- [ ] Datos verificados
- [ ] CTA incluido
- [ ] Hashtags relevantes

### Imagen:
- [ ] Calidad adecuada
- [ ] Formato correcto
- [ ] Sin marcas de agua
- [ ] Optimizada para web

### Programación:
- [ ] Hora óptima de publicación
- [ ] Plataforma correcta
- [ ] Frequencia adecuada
- [ ] Seguimiento programado

## Métricas de Calidad

### Para cada pieza:
- [ ] Consistencia con marca (100%)
- [ ] Sin errores ortográficos
- [ ] Disclaimer incluido (si aplica)
- [ ] USD/m² incluido (si aplica)
- [ ] Datos de contacto correctos
- [ ] Optimizada para plataforma
- [ ] Responsive (si aplica)
- [ ] Profesional y elegante
