### Reglas de Color
1. **NUNCA** usar colores fuera de esta paleta
2. **SIEMPRE** mantener contraste WCAG AA (4.5:1 mínimo)
3. **SIEMPRE** usar dorado como acento, no como color principal
4. **NUNCA** usar más de 3 colores por pieza
5. **SIEMPRE** usar negro para fondos oscuros
6. **SIEMPRE** usar blanco para texto en fondos oscuros

6. **NUNCA** justificar texto (mejor alineado a la izquierda)

## Imágenes

### Fotos Reales (Estado de Obra)
- **USAR**: Para mostrar avance real
- **INCLUIR**: Fecha y estado
- **FIABLE**: Transmite honestidad

### Calidad
- **SIEMPRE** usar imágenes de alta resolución
- **NUNCA** estirar o distorsionar
- **SIEMPRE** mantener proporciones
- **OPTIMIZAR** para web (máx 200KB por imagen)

## Layout y Espacio

### Principios
1. **Espacio negativo**: Que respire, no llenar todo
2. **Sin superposiciones**: Elementos claros y separados
3. **Jerarquía**: Lo más importante primero
4. **Alineación**: Grid o flexbox, nunca al ojo

### Espaciado
| Elemento | Espaciado |
|----------|-----------|
| Secciones | 40-50px |
| Entre elementos | 20-30px |
| Interno de cards | 25-35px |
| Bordes de página | 50px |

### Grid
- **Brochure**: 2 columnas (50/50 o 60/40)
- **Flyer**: 1 columna (centrado)
- **Email**: 1 columna (600px max)

## Contenido

### Copy
- **Tono**: Premium, profesional, directo
- **Extensión**: Corto y conciso
- **Datos**: Siempre con fuente (USD/m², metros, precios)
- **CTA**: Claro y accionable

### Datos Obligatorios
1. Precio (USD)
2. USD/m²
3. Superficie (m²)
4. Ubicación
5. Contacto
6. Disclaimer (en renders)

### Datos Prohibidos
1. Adjetivos vacíos ("increíble", "espectacular")
2. Promesas no documentadas
3. Datos sin verificar
4. Información engañosa

## Formatos por Plataforma

### WhatsApp (Envío)
- **Ancho**: 750px
- **Formato**: HTML single-file
- **Imágenes**: Inline o rutas relativas
- **Responsive**: Móvil primero

### Instagram
- **Tamaño**: 1080x1080px
- **Formato**: PNG o JPG
- **Texto**: Mínimo (regla del 20%)
- **CTA**: En copy, no en imagen

### Email
- **Ancho**: 600px máximo
- **Responsive**: Siempre
- **Imágenes**: Inline o CDN
- **Botones**: 44px mínimo

### Impresión
- **Tamaño**: A4 (210x297mm)
- **Resolución**: 300dpi
- **Colores**: CMYK (convertir de RGB)
- **Sangrado**: 3mm

## Prohibiciones Absolutas

### NUNCA hacer:
1. Usar look genérico de IA (cream+serif+terracotta)
2. Usar negro + verde ácido
3. Superponer texto en imagen sin contraste
4. Usar más de 2 fuentes
5. Llenar todo el espacio (sin espacio negativo)
6. Enviar sin disclaimer de renders
7. Modificar precios sin autorización
8. Usar colores fuera de paleta
