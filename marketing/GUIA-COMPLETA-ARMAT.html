Cómo Funciona el Sistema

Flujo de trabajo del CEO orquestador con 16 agentes especializados

Flujo General

1
Vos hacés una pregunta en lenguaje natural (ej: "Necesito una propuesta para vender departamentos en Centenario")
▼
2
El CEO analiza la petición y decide: ¿es simple (la resuelvo yo) o compleja (delego a un agente)?
▼
3
El CEO delega al agente especializado correcto con contexto completo
▼
4
El agente ejecuta la tarea usando sus skills y instrucciones específicas
▼
5
El CEO revisa la calidad y presenta el resultado
▼
6
Recibís el resultado y podés pedir ajustes

Framework de Asesoramiento Inmobiliario

Cada análisis sigue 4 pasos: 1) Diagnóstico → situación actual del mercado, problemas, oportunidades. 2) Estructura → organizativa, legal, comercial. 3) Plan de acción → prioridades, cronograma, responsables. 4) Riesgos → mitigación, contingencias.

Ejemplos Reales de Uso

Cómo funcionan los agentes en situaciones concretas de real estate

Usuario pregunta

"Armame un brochure profesional para vender los departamentos de ENTANYQ"

Sistema ejecuta

El CEO delega a 2 agentes:

1. Marketing → Selecciona renders, estructura brochure, escribe copy profesional
2. Marketing → Genera HTML con estilo corporativo, galería, datos de unidades

Agentes usados: CEO → Marketing
Usuario pregunta

"¿Cuánto me costaría construir un edificio de 20 departamentos en Centenario?"

Sistema ejecuta

El CEO delega a 3 agentes:

1. Obras → Estima costos de construcción por m², materiales, mano de obra
2. Presupuestos → Arma presupuesto detallado con desglose
3. Finanzas → Proyecta ROI, punto de equilibrio, escenarios

Agentes usados: CEO → Obras → Presupuestos → Finanzas
Usuario pregunta

"Necesito una propuesta de venta para un cliente que quiere un departamento de 1 dormitorio"

Sistema ejecuta

El CEO delega a 2 agentes:

1. Comercial → Verifica disponibilidad, calcula financiamiento, genera propuesta
2. Comercial → Reserva unidad, prepara contrato

Agentes usados: CEO → Comercial

Cómo Invocar los Agentes

Formas de usar el sistema y mapa de delegación del CEO

Forma 1: Hablarle al CEO (recomendado)

// Simplemente hacé tu pregunta y el CEO orquesta todo "Armame un brochure para ENTANYQ" "¿Cuánto cuesta construir un departamento de 1 dormitorio?" "Necesito una propuesta comercial para este cliente" "Creame un post de Instagram del avance de obra" "Revisá si este contrato tiene riesgos legales"

Forma 2: Invocar un agente directo

// Si sabés qué agente necesitás, invocalo directo armat-ceo → Orquestador general armat-comercial → Ventas, pricing, propuestas armat-marketing → Contenido, branding, redes armat-finanzas → Presupuestos, ROI, proyecciones armat-legal → Contratos, permisos armat-obras → Construcción, avance armat-data → KPIs, dashboards, reportes armat-compras → Materiales, proveedores

Delegación en paralelo (tareas independientes)

// Cuando las tareas NO dependen entre sí, lanza todas juntas task(subagent_type: "armat-comercial", description: "Propuesta", prompt: "...") task(subagent_type: "armat-marketing", description: "Brochure", prompt: "...") task(subagent_type: "armat-finanzas", description: "Análisis", prompt: "...")
