Casos de Uso Reales

Situaciones de negocio y como el sistema las resuelve

Caso 1: Cliente quiere comprar con credito
Vos escribis

Tengo un cliente que quiere el 1 dormitorio pero con credito hipotecario. Como lo proceso?

Sistema ejecuta

CEO - Comercial + Finanzas: Genera propuesta con precio contado USD 85.000, simula credito a 20 anos con tasa del 7%, calcula cuota mensual ~USD 650, compara con opcion de financiamiento directo (50% + 12 cuotas).

Agentes: CEO - Comercial - Finanzas · Tiempo: 8 segundos
Caso 2: Calcular materiales de un departamento
Vos escribis

Necesito saber cuanto material necesito para el monoambiente. Te paso las medidas: 5m x 4.50m, banos 2m x 1.80m

Sistema ejecuta

CEO - Obras: Calcula materiales con rendimientos Neuquen: 15 ladrillos/m2, 0.015 m3 hormigon/m2, ceramica con 8% desperdicio. Genera lista completa con costos en USD y ARS.

Agentes: CEO - Obras · Output: Tabla de materiales + costos
Caso 3: Crear brochure para nuevo proyecto
Vos escribis

Arrancamos un proyecto nuevo en Neuquen Capital. Necesito un brochure para vender

Sistema ejecuta

CEO - Marketing: Crea estructura de proyecto, genera brochure HTML profesional con renders, datos, formas de pago. Crea version autocontenido (base64) para enviar por WhatsApp.

Agentes: CEO - Marketing · Output: 2 archivos HTML listos para usar
Caso 4: Reporte de avance para cliente
Vos escribis

El cliente de la unidad 3 pregunta como va la obra. Necesito un reporte

Sistema ejecuta

CEO - Obras + Calidad: Genera reporte con % avance fisico (65%), fotos del estado actual, checklist de calidad verificado, proximos pasos y fecha estimada de entrega.

Agentes: CEO - Obras - Calidad · Output: Reporte PDF + fotos

Preguntas Frecuentes

Las dudas mas comunes antes de empezar

Necesito saber de IA o programacion?
No. ARMAT esta disenado para que hables en espanol natural, como le hablarias a un empleado. No hay comandos, no hay codigo, no hay configuracion tecnica.
Necesito pagar por ChatGPT Plus o similar?
No. El sistema usa modelos de IA gratuitos (MiMo V2.5 Free) que son suficientes para el 90% de las tareas. No necesitas suscripciones adicionales.
Que pasa si cambia OpenAI o el modelo de IA?
ARMAT esta disenado para ser flexible. Los agentes funcionan con cualquier modelo de IA compatible. Si un modelo deja de estar disponible, se cambia con una linea de configuracion.
Puedo agregar mis propios agentes o skills?
Si. El sistema es modular. Podes crear nuevos agentes, agregar skills a los existentes, o modificar los que ya estan. Cada agente es un conjunto de archivos de texto.
Funciona con cualquier tipo de empresa?
ARMAT esta optimizado para empresas inmobiliarias, pero los agentes base (CEO, Comercial, Marketing, Finanzas, Legal, Admin, Data) funcionan en cualquier industria. Los agentes de obra y construccion son especificos del sector.
Cuanto tarda en estar funcionando?
15-30 minutos para tener el sistema basico funcionando. La instalacion completa con todos los agentes y skills toma entre 30 y 45 minutos.
Necesita un servidor o infraestructura especial?
No. Funciona en tu computadora. No hay servidores que mantener, no hay hosting que pagar, no hay bases de datos que administrar.
Quien mantiene y actualiza el sistema?
ARMAT incluye actualizaciones continuas. Nuevos skills, mejores agentes, mas integraciones. Todo se actualiza con un comando simple. Ademas, tenes soporte para personalizar el sistema a tu medida.
Mis datos estan seguros?
Si. Todo funciona en tu computadora. No se envian datos a servidores externos. La informacion de tus clientes, contratos y proyectos queda en tu control total.
