# Workflow: Control de Obra

## Descripción
Flujo diario para monitorear avance de construcción, calidad y presupuesto.

## Trigger
- Diario a las 7:00 AM
- Evento de calidad (no conformidad)

## Pasos

### 1. Bitácora Diaria (Obras)
- Registrar actividades del día
- Documentar personal y materiales
- Fotografiar avance

### 2. Verificación de Calidad (Calidad)
- Inspeccionar áreas trabajadas
- Verificar especificaciones
- Documentar hallazgos

### 3. Control Presupuestario (Presupuestos)
- Actualizar avance financiero
- Comparar con presupuesto
- Identificar variaciones

### 4. Reporte Ejecutivo (Data)
- Consolidar datos
- Generar dashboard
- Enviar a dirección

## Agentes Involucrados
- Obras
- Calidad
- Presupuestos
- Data

## SLA
- Bitácora: 8:00 AM
- Inspección: 10:00 AM
- Reporte: 5:00 PM
