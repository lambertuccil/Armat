# Base de Conocimiento — Ventas Inmobiliarias

## Técnicas de Venta

### 1. Venta Consultiva
- Escuchar antes de hablar
- Hacer preguntas abiertas
- Identificar necesidades reales
- Ofrecer soluciones, no productos
- Documentar todo

### 2. Manejo de Objeciones
| Objeción | Respuesta |
|----------|-----------|
| "Es caro" | Mostrar USD/m² vs mercado |
| "Está lejos" | Enfatizar crecimiento zona |
| "Está en gris" | Mostrar avance y cronograma |
| "No tengo tiempo" | Ofrecer financiamiento |

### 3. Cierre
- Identificar señales de compra
- Crear urgencia (50% vendido)
- Ofrecer siguiente paso claro
- Documentar acuerdo

## Perfil del Cliente

### Inversores
- Buscan retorno de inversión
- Les interesa Vaca Muerta
- Valorizan datos financieros
- Deciden rápido

### Familias
- Buscan espacio y comodidad
- Les interesa la zona
- Valorizan calidad de vida
- Deciden con calma

### Primera Compra
- Buscan accesibilidad
- Les interesa financiamiento
- Valorizan asesoramiento
- Tienen dudas frecuentes

## Pipeline de Ventas

### Etapas
1. **Lead nuevo**: Consulta inicial
2. **Interesado**: Muestra interés
3. **Calificado**: Confirma capacidad
4. **Propuesta**: Recibe propuesta
5. **Negociación**: Ajusta condiciones
6. **Cierre**: Firma contrato
7. **Post-venta**: Seguimiento

### Métricas
- Tasa de conversión
- Tiempo promedio de cierre
- Ticket promedio
- Satisfacción del cliente
