## 3. PROCESO DE VENTA

### 3.1 Script de atención inicial

**WhatsApp/Instagram:**
```
"¡Hola! Gracias por contactarnos sobre ENTANYQ. 
¿Te gustaría recibir información sobre los departamentos disponibles?"
```

**Si dice sí:**
```
"Perfecto. Tenemos monoambientes desde USD 60.000 y departamentos de 
1 dormitorio desde USD 85.000. ¿Cuál te interesa más?"
```

**Agendar visita:**
```
"¿Te gustaría conocer la obra? Tenemos disponibilidad el [FECHA] 
a las [HORA]. ¿Te sirve?"
```

### 3.2 Guion de visita

1. **Bienvenida** (5 min) — Presentación, café, ambiente
2. **Recorrido** (20 min) — Mostrar avance de obra, explicar acabados
3. **Datos** (10 min) — Superficie, orientación, balcón, cocheras
4. **Financiamiento** (10 min) — Opciones de pago, cuotas
5. **Cierre** (5 min) — Reservar, dejar material

### 3.3 Manejo de objeciones

| Objeción | Respuesta |
|---|---|
| "Es muy caro" | "Comparado con otros emprendimientos, estamos 10-15% abajo. Además, tenés financiación en cuotas fijas USD." |
| "No conozco ARMAT" | "Somos una desarrolladora joven con más de 10 años de experiencia. ¿Te gustaría ver otros proyectos que hicimos?" |
| "¿Y si no se termina?" | "Tenés contrato, memoria descriptiva, y ya vendimos el 50%. El edificio está en construcción." |
| "Necesito pensarlo" | "Entiendo. ¿Qué te gustaría que te aclare para ayudarte a decidir?" |
| "No tengo tiempo ahora" | "¿Te parece si te envío la información por WhatsApp y coordinamos una visita para la semana que viene?" |
