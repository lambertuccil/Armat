# Armat - MCP Tools & Skills

## MCP Servers

### nucleus
Herramientas de gobierno y seguridad.

| Tool | Descripción |
|------|-------------|
| `governance_set_mode` | Establecer modo IDE |
| `sessions_start` | Iniciar sesión |
| `sessions_register` | Registrar sesión |

### context7
Documentación actualizada de librerías.

| Tool | Descripción |
|------|-------------|
| `resolve-library-id` | Resolver ID de librería |
| `query-docs` | Consultar documentación |

### engram
Memoria persistente entre sesiones.

| Tool | Descripción |
|------|-------------|
| `mem_save` | Guardar observación |
| `mem_search` | Buscar en memoria |
| `mem_context` | Contexto reciente |

### duckdb
Base de datos para análisis financiero y de proyectos.

| Tool | Descripción |
|------|-------------|
| `query` | Ejecutar SQL |
